# What is SAP BTP

## Definition
SAP BTP (Business Technology Platform) is a cloud platform used to build, integrate, and extend applications.

## Why it matters
It allows connecting SAP with other systems and building new apps without modifying core SAP.

## Real Example
Connecting SAP with Salesforce using Integration Suite.

## Key Points
- Cloud platform
- Supports integration and extension
- Keeps SAP core clean
- Includes Integration Suite, API Management
