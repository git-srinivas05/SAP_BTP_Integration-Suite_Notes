# What is SAP
:
 Definition
SAP is an enterprise software used to manage business processes in a centralized system.

## Why it matters
Large companies use SAP to handle finance, sales, inventory, and HR in one system.

## Real Example
When you order a product:
- Sales creates order
- Warehouse checks stock
- Finance generates invoice  
All happen inside SAP.

## Key Points
- Centralized system
- Handles multiple business processes
- Used by large enterprises
- Core system = S/4HANA
