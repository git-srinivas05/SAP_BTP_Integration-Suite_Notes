# API vs Web Service

## Definition
API is a general interface for communication.  
Web Service is a type of API that works over the web.

## Why it matters
Helps understand REST vs SOAP in SAP integrations.

## Real Example
REST API → modern applications  
SOAP Web Service → older SAP systems

## Key Points
- All web services are APIs
- Not all APIs are web services
- SOAP → XML-based
- REST → JSON-based
