# XML

## Definition
XML (Extensible Markup Language) is a structured data format used in older systems.

## Why it matters
SAP systems often use XML (IDoc, SOAP).

## Real Example
<order>
  <orderId>101</orderId>
  <customer>Srinivas</customer>
</order>

## Key Points
- Tag-based structure
- More verbose than JSON
- Used in SOAP and SAP legacy systems
