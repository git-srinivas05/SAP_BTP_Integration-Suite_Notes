# JSON vs XML

## Definition
Comparison between two data formats used in integrations.

## Why it matters
Helps choose correct format in real integration scenarios.

## Real Example
Same data in JSON and XML formats

##JSON
{
  "orderId": 101,
  "customer": "Srinivas"
}

##XML 
<order>
  <orderId>101</orderId>
  <customer>Srinivas</customer>
</order>

## Key Points
- JSON → lightweight, faster
- XML → structured, strict
- JSON → modern APIs
- XML → legacy SAP systems
